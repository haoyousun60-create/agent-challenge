# Nosana x ElizaOS Challenge Submission

## Project: Crypto Portfolio Monitor Agent 🤖💰

### Agent Description (≤300 words)

Crypto Portfolio Monitor Agent is a personal AI assistant built with ElizaOS framework, designed specifically for cryptocurrency enthusiasts who want to stay on top of their investments 24/7.

**Core Capabilities:**
- Real-time price monitoring for DOGE, ADA, XRP, BTC, ETH, and SOL
- Price alerts (notify when coins drop below or rise above thresholds)
- Portfolio tracking with 24h change indicators
- Market news aggregation

**What Makes This Agent Unique:**
This agent is tailored for the Chinese crypto community with Bilingual support and understanding of specific coins (DOGE, ADA, XRP) that老板 cares about. It uses the free CoinGecko API so anyone can run it without API keys.

**Technical Stack:**
- ElizaOS v2 Framework
- TypeScript
- CoinGecko API (free, no key required)
- Nosana decentralized GPU network

**Use Cases:**
1. "What's the price of DOGE?" → Instant response
2. "Alert me when ADA drops below $0.50" → Alert set
3. "Show me my portfolio" → Full overview with 24h changes
